LoveBot is an AI-powered prompt generator designed specifically for dating apps, with support for Hinge and Bumble. I created this project as a fun way to experiment with the then new OpenAI API.

Please note: The deployment may be unstable as the Supabase project occasionally goes dormant. If you find the completions aren't working when you try to use it, this is likely the reason.
